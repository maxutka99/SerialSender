# SerialSender
Python program to send 1 or 0 to serial port
# Installing
Installation guide
```bash
git clone https://github.com/maxutka99/SerialSender.git
cd SerialSender
pip3 install -r requriements.txt
```

# Running
```bash
python3 serial_sender_gui.py
```
